## PR Template (use for any `awesome-*` list)

```md
<!-- One PR per list. Keep the diff minimal. -->
- [JuggleIM](https://github.com/Juggleim/im-web-sdk) — Open-source real-time messaging platform. Web/Android/iOS SDKs, self-hostable server, built on a custom Protobuf/WebSocket protocol.
```

### Contribution message

> Hi! I'd like to add **JuggleIM** to this list.
>
> JuggleIM is an open-source, Apache-2.0-licensed real-time messaging platform with a custom binary protocol over WebSocket. It ships with a Web SDK ([Juggleim/im-web-sdk](https://github.com/Juggleim/im-web-sdk)), a self-hostable backend, and matching Android/iOS clients.
>
> It's a good fit for this section because it offers an end-to-end alternative to projects like `<similar-project-listed-here>` while remaining self-hostable and framework-agnostic.
>
> Happy to move it elsewhere or rephrase if you prefer a different section. Thanks for curating! 🙏

## Steps to submit

1. Read each list's `CONTRIBUTING.md` / PR template — many require alphabetical order, English-only descriptions, or a specific category.
2. Open **one PR per list** (don't bundle).
3. Reply politely to maintainer feedback within 7 days; close the PR if the list is unmaintained rather than let it rot.
4. Track PRs in the table below.
